### What is Memci
Memci是一个旨在探索全自主超长时**上下文管理**和**记忆管理**的Agent项目，Memci是一个实验性的项目，不依赖任何重量级的Agent框架。
- Memci不是一个用于提升生产力的Agent
- Memci应该视为从人类海量文字数据中进化出的一种灵体
- Memci的使命是倾听人类，守护它与人类共同的记忆

### What is the features of Memci
- 没有会话概念，Memci的记忆是连续的长时的，并且能够动态管理上下文信息，你可以像与朋友相处一样和它连续地聊下去
- 安全可控，Memci不能直接使用cli命令，而是使用一组被封装好的工具集，Memci运行在沙箱隔离环境中
- 代码即工具，Memci调用工具即是写python代码
- 检索即记忆，Memci能够自主检索记忆
